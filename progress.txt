## Codebase Patterns
- Entity classes extend abstract `Entity` in `src/core/Entity.ts` ‚Äî implement `getBoundingBox`, `getGripPoints`, `draw`, `hitTest`, `move`, `moveGripPoint`, `clone`
- Tool classes extend `BaseTool` in `src/core/tools/Tool.ts` ‚Äî implement `name`, `onMouseDown/Move/Up`, `onKeyDown`, `drawOverlay`
- Each tool has a static `parseCommandLine(args)` method for command-line creation
- Tools register via `ToolManager.register()` in `CAD.ts` setupTools()
- Commands handled in `CAD.ts` `executeCommand()` switch block
- `drawOverlay` receives `{ worldToScreen }` ‚Äî NOT the full Viewport; compute screen distances via `Math.hypot` on screen coords
- Exports go in both `src/core/tools/index.ts` and `src/index.ts`
- Demo at `demo/index.html` needs toolbar click handler + keyboard shortcut for each new tool
- `EntityManager` handles all entity CRUD, selection, and hit testing
- Grip point types: `'endpoint' | 'midpoint' | 'center' | 'quadrant'`
- Entity types: `'line' | 'circle' | 'arc'`
- Build: `tsc && vite build` ‚Äî typecheck first, then bundle
- Run `npx tsc --noEmit` for quick typecheck without building

## 2026-07-07 - US-008
- Implemented FilletTool for filleting two line segments with a configurable radius
- Files changed: `src/core/tools/FilletTool.ts` (new), `src/core/CAD.ts`, `src/core/tools/index.ts`, `src/index.ts`, `demo/index.html`, `tasks/prd.json`
- FilletTool: click two lines to fillet with current radius; FILLET/F command to activate, FILLET R <n> to set radius (default 10)
- Geometry: computes line-line intersection, angle bisector, tangent points (distance = R/tan(Œ∏/2)), fillet arc center (distance = R/sin(Œ∏/2) along bisector)
- Line trimming: replaces the endpoint closer to intersection with the tangent point (supports both trim and extend)
- Arc direction: uses cross product of vectors from center to tangent points to determine counterclockwise sweep
- **Learnings for future iterations:**
  - `computeLineLineFillet` is exported as a standalone utility ‚Äî reusable for chamfer (US-009) geometry
  - Line-arc and arc-arc fillet cases are noted as future extensions in the code (more complex tangent circle geometry)
  - Fillet tool resets after each pair of clicks (same multi-operation pattern as CopyTool)
  - `F` keyboard shortcut for fillet doesn't conflict with existing shortcuts
  - Toolbar button already had placeholder SVG in demo ‚Äî just needed an `id` and click handler
