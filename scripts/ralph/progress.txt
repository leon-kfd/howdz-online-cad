## Codebase Patterns
- Project uses Vite + TypeScript in library mode (src/index.ts as entry)
- Canvas rendering uses requestAnimationFrame loop in Renderer class
- Viewport handles all coordinate transforms (screenToWorld/worldToScreen)
- Entity base class with abstract methods for draw/hitTest/move/clone
- Grid uses niceNumber() for 1-2-5 adaptive spacing
- Build: `tsc && vite build`; typecheck: `npx tsc --noEmit`
- Status bar created in CAD.ts createDOM() method
- Entity types defined in src/core/types.ts and src/core/Entity.ts
- Tool system: Tool interface → BaseTool → concrete tools (SelectTool, LineTool)
- Tools receive ToolContext with mouseWorld, mouseScreen, orthoMode, scale
- ToolManager dispatches events to active tool, manages tool registration
- EntityManager manages entity collection + selection state (Set of IDs)
- Renderer draws: grid → axes → entities → crosshair → tool overlay
- Command line: CAD.executeCommand() parses "CMD args" format
- TypeScript strict: prefix unused params with _, use 'export type' for isolatedModules
